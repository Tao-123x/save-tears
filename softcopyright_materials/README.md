# 软著申报材料目录

本目录基于当前仓库代码实际实现情况整理，用于辅助准备中国计算机软件著作权登记材料。

## 文件说明

- `SOFTWARE_MANUAL.md`
  - 软件说明书草稿，可作为说明文档主体。
- `SOURCE_CODE_SUBMISSION_GUIDE.md`
  - 源程序提交规则、当前项目源码统计、建议提交范围。
- `APPLICATION_INFO_TEMPLATE.md`
  - 申请表填写模板，包含建议值和待补字段。

## 建议使用顺序

1. 先看 `APPLICATION_INFO_TEMPLATE.md`
   - 把著作权人、完成日期、发表状态等信息补齐。
2. 再看 `SOFTWARE_MANUAL.md`
   - 作为软件说明书基础文本，后续可转为 Word。
3. 最后看 `SOURCE_CODE_SUBMISSION_GUIDE.md`
   - 按里面的文件顺序整理源码 PDF。

## 当前结论

- 当前项目适合申报为：`Save Tears 节水管理系统 V1.0`
- 当前核心源码统计约为：`18` 个文件，`1915` 行
- 按《计算机软件著作权登记办法》第十条，这个体量通常建议直接提交全部核心源程序，而不是只交前后各 30 页
