# 申请表填写模板

本文件用于整理软著申报表中的关键信息。带 `待确认` 的字段需要你自己补充或确认。

## 1. 软件基本信息

- 软件全称：`Save Tears 节水管理系统`
- 软件简称：`Save Tears`
- 版本号：`V1.0`
- 分类建议：`应用软件`
- 软件说明：`面向宿舍、公寓等场景的节水管理系统`

可选备选名：

1. `基于微信小程序的宿舍节水管理系统`
2. `Save Tears 用水监测与账单管理系统`

建议：

优先使用不超过 20 个汉字且业务边界清晰的名称，避免把“智能”“大数据”“AI”写进正式名称，除非代码中已有充分对应实现。

## 5. 软件功能简介

可直接用于申请表中的功能简介：

本软件是一套基于微信小程序和 Python 后端的节水管理系统，主要面向宿舍、公寓等按房间管理的场景。系统实现了用户注册登录、房间号绑定、用水数据管理、污水浊度管理、水费账单管理等功能。软件采用 uni-app、Vue 3、FastAPI、SQLAlchemy 等技术开发，支持 SQLite 或 MySQL 数据存储，可实现房间维度的基础用水信息查询与管理。

## 6. 技术特点简介

可直接用于技术特点栏：

软件采用前后端分离架构，小程序端负责用户交互与信息展示，后端负责接口服务和数据持久化。系统通过房间号统一关联用户、用水、污水和账单信息，具备结构清晰、部署简单、易扩展等特点。
